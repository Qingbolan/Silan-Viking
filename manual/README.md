## Screenshots / 截图

`images/` holds the screenshots the manuals embed. `images/README.md` is the
capture checklist — run each command and drop the screenshot in. The manuals
read fine before the images are added; screenshots are an enhancement.
`images/` 存放手册引用的截图,`images/README.md` 是截图清单 —— 跑命令、把截图
放进去即可。截图补齐前手册也完整可读,截图是增强项。
